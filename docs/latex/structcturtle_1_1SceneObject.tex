\doxysubsection{Detailed Description}
Turtles append Scene Objects to a list to keep track of what it has drawn (a history). \mbox{\hyperlink{structcturtle_1_1SceneObject}{Scene\+Object}} holds a description of something that needs to be on the screen. It\textquotesingle{}s a general object which encompasses ALL things that can be on screen, ranging from stamps, misc. geometry, and strings.
